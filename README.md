#Bill

##### 软件介绍
>  也许你不是一个爱计较的人，但每个月花的钱总是让你很郁闷，我的钱到底花拿去了。还是我的钱是不是丢过，不然怎么花了这么多。但是有了Bill后，你再也不用烦恼了，Bill是一款简便的记账app，UI简便清晰，让你的记账不再那么繁琐，轻松记录每一天的账单。 

#####主要功能
- 根据每天的日期来记账 
- 根据日期来查询您记过的账单 
- 自动计算你一天的消费情况
-  有四个账本供您选择，让记账更人性化
